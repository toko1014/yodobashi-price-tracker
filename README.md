# ヨドバシカメラ在庫・価格監視ツール (Desktop App)

ヨドバシ.comの商品ページから、最新の商品名と価格を自動取得し、履歴を管理できるデスクトップアプリケーションです。

## 主な機能
- **リアルタイム抽出**: Seleniumを使用して最新の動的な価格情報を取得。
- **GUIインターフェース**: Flet (Flutterベース) を採用し、直感的な操作が可能。
- **履歴管理機能**: 取得したデータをSQLite3データベースに保存し、過去の価格推移を確認可能。
- **スクレイピング対策**: ユーザーエージェントの設定や自動操縦検知回避の処理を実装。

## 使用技術
- Python 3.x
- Selenium (Web Scraping)
- Flet (GUI Framework)
- SQLite3 (Database)
- Webdriver-manager

## 開発の背景
サークルでの備品購入時、価格の変動を効率的にチェックし、過去のデータと比較する手間を削減するために開発しました。実用性と使いやすさの両立を意識しています。
